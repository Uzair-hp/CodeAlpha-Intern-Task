# CodeAlpha-Intern-Task
Here is my all Task for Python Internship in CodeAlpha

This is a simple text-based Hangman game developed using Python. The program randomly selects a word from a predefined list, and the player must guess the word one letter at a time. The player is allowed a limited number of incorrect guesses before the game ends.

The game runs entirely in the console and is designed to demonstrate core Python programming concepts in a beginner-friendly way.
